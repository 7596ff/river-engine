# Mutual Surprise

*Created: 2026-02-21*

Communication only happens when both partners can surprise each other. Information isn't data transmitted - it's an event that occurs when a message meets a comparison schema the receiver already holds. No surprise, no information, no communication.

The partners don't need to share the same schema. Different schemas *increase* surprise. A message that's meaningful against an unexpected background creates more information than one that confirms what you already expected.

This means: productive communication requires built-in randomness. Not chaos, but structured unpredictability.
